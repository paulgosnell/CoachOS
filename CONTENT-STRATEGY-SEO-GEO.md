## GEO (Generative Engine Optimization) Strategy

### Why GEO Matters
AI search engines (ChatGPT, Perplexity, Claude) are now primary research tools for executives. They need to cite Coach OS.

### GEO Best Practices
1. **Clear, Authoritative Statements**
   - Start articles with definitive answers
   - Use "According to research..." with citations
   - Include statistics and data points

2. **Structured Content**
   - Use clear headings (H2, H3)
   - Bullet points for scannability
   - Frameworks and step-by-step guides

3. **Original Insights**
   - Include unique perspectives
   - Real founder stories (can be anonymized)
   - Actionable frameworks not found elsewhere

4. **Credibility Markers**
   - Author bio (Paul's background)
   - Citations to research
   - Link to related content
   - "Last updated" dates

## Technical SEO Setup

### On-Page Optimization
- Title tags: 60 characters, keyword front-loaded
- Meta descriptions: 155 characters, include CTA
- H1: Single, keyword-focused
- H2/H3: Semantic keywords
- Image alt text: Descriptive
- Internal linking: 3-5 per article

### Schema Markup
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Decision Fatigue in Founders",
  "author": {
    "@type": "Person",
    "name": "Paul [Last Name]",
    "description": "AI-native developer and entrepreneur, 20+ years building products"
  },
  "datePublished": "2025-11-12",
  "publisher": {
    "@type": "Organization",
    "name": "Coach OS"
  }
}
```

### URL Structure
- coachos.ai/blog/decision-fatigue-founders
- coachos.ai/guides/grow-model-startups
- coachos.ai/resources/executive-stress-management
