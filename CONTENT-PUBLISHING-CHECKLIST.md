# Content Publishing Checklist

**Use this checklist for every article you publish**

---

## Pre-Publication

### Content Quality
- [ ] Article is 1,500+ words
- [ ] Personal anecdotes or real examples included
- [ ] Actionable takeaways provided
- [ ] No jargon without explanation
- [ ] Conversational, authentic voice
- [ ] Proofread for typos and grammar

### SEO Optimization
- [ ] Target keyword in title
- [ ] Target keyword in first paragraph
- [ ] Target keyword in at least one H2
- [ ] Related keywords naturally included
- [ ] Internal links to 2-3 other articles (when available)
- [ ] External links to 2-3 authoritative sources
- [ ] Meta description written (155 characters)
- [ ] URL slug is clean and keyword-focused

### Structure
- [ ] Clear H1 title
- [ ] H2 subheadings every 300-400 words
- [ ] Short paragraphs (2-4 sentences max)
- [ ] Bullet points for scannability
- [ ] Bold key phrases (not excessive)
- [ ] Images with alt text (if applicable)

### GEO Optimization
- [ ] Clear, definitive statements early
- [ ] Statistics with citations
- [ ] Frameworks or step-by-step guides
- [ ] Unique insights not found elsewhere
- [ ] Author bio at end
- [ ] Last updated date at top

---

## Publication Process

### Blog
- [ ] Create post in your CMS
- [ ] Add featured image (1200x630px)
- [ ] Set publish date and time
- [ ] Add tags (5-7 relevant tags)
- [ ] Add to category (coaching, founders, productivity, etc.)
- [ ] Enable comments
- [ ] Submit URL to Google Search Console after publishing

### Medium
- [ ] Wait 7 days after blog publication
- [ ] Copy article to Medium
- [ ] Add canonical link pointing to your blog
- [ ] Add "Originally published at coachos.ai/blog/..." at top
- [ ] Add appropriate tags (5 max)
- [ ] Submit to relevant publications (The Startup, Entrepreneurship)
- [ ] Add call-to-action at end with Coach OS link

### LinkedIn
- [ ] Create as article (not post)
- [ ] Add personal intro paragraph about why you wrote this
- [ ] Keep same content
- [ ] Add "Read the full version with resources at coachos.ai"
- [ ] Post from your personal profile
- [ ] Share the article as a regular post with commentary
- [ ] Engage with comments for first 48 hours

---

## Post-Publication

### Distribution (Day 1)
- [ ] Share on LinkedIn (personal profile)
- [ ] Tweet thread with key points (link in final tweet)
- [ ] Share in relevant Slack communities (if appropriate)
- [ ] Send to email newsletter subscribers
- [ ] Share on any founder Telegram/Discord groups you're in
- [ ] Post in relevant subreddits (carefully, follow rules)

### Monitoring (Week 1)
- [ ] Check Google Search Console for indexing
- [ ] Monitor social shares and comments
- [ ] Respond to all comments within 24 hours
- [ ] Track traffic in Google Analytics
- [ ] Note which distribution channels drove traffic
- [ ] Check for backlinks (Ahrefs or similar)

### Repurposing (Week 2)
- [ ] Create LinkedIn carousel from key points
- [ ] Pull quotes for Twitter/LinkedIn posts
- [ ] Record short video summary (Loom)
- [ ] Create 1-page PDF summary for lead magnet
- [ ] Identify elements for future articles
- [ ] Add to email nurture sequence

## SEO Checklist (Technical)

### Meta Tags
```html
<title>Article Title | Coach OS</title>
<meta name="description" content="Your 155-character meta description with target keyword">
<meta name="keywords" content="founder burnout, executive coaching, on-demand coaching">
<meta name="author" content="Paul - Founder, Coach OS">
<link rel="canonical" href="https://coachos.ai/blog/article-slug">
```

## Success Metrics

Track these for each article:

### Traffic (Week 1)
- Pageviews
- Unique visitors
- Average time on page (aim for 3+ minutes)
- Bounce rate (aim for <60%)
- Traffic sources (organic, social, direct)

### Engagement (Week 1)
- Social shares
- Comments
- Backlinks
- Email signups
- Waitlist signups

### SEO (Month 1)
- Google Search Console impressions
- Click-through rate
- Average position for target keyword
- Pages linking to this article
- Total backlinks

### Long-term (Month 3)
- Organic traffic trend
- Keyword rankings (top 10, 20, 50)
- Domain authority impact
- Conversion rate (visitor to signup)

## Red Flags

Don't publish if:
- [ ] Article feels generic or templated
- [ ] No personal perspective or story
- [ ] Keyword stuffed or unnatural
- [ ] Copied content from elsewhere
- [ ] No actionable takeaways
- [ ] Errors in facts or figures
- [ ] Missing author bio or CTA
- [ ] Not mobile-friendly

Fix these before going live.

---

## Emergency Fixes

If article performs poorly (low traffic/engagement after 48 hours):

1. **Headline Problem**
   - A/B test new headline on social
   - Make it more specific or emotional
   - Add numbers or urgency

2. **Distribution Problem**
   - Reshare with different angle
   - Try different platforms
   - Ask peers to share

3. **Content Problem**
   - Add more personal stories
   - Include more actionable tips
   - Improve formatting for scannability

4. **SEO Problem**
   - Check if indexed (Google Search Console)
   - Improve title tag and meta description
   - Add more internal/external links

---

**Remember:** Every article is a long-term asset. It'll drive traffic for years if optimized properly. Don't rush publication to hit a date - get it right.

**Minimum Standard:** Would I share this with a founder friend who's struggling? If not, keep editing.
